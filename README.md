This repository is my first dedicated project where I use a real dataset to create queries to showcase and document my current level of proficiency in SQL. 

Dataset Information (record-specific data can be found in the data folder): 
Source: Kaggle
Name: Video Game Sales 
Columns: ranking, game name, platform, year, genre, publisher, na_sales, eu_sales, jp_sales, other_sales, global_sales

Analysis performed/list of queries: 
- Genre sales performance
- Publisher market share
- Regional sales dominance
- Top game per genre
- Window function ranking
- Running sales totals

Environment: PostgreSQL (queries executed using pgAdmin)
